# thermal-comfort

## Installation

via https

```bash
pip install git+https://github.com/RUBclim/thermal-comfort
```

via ssh

```bash
pip install git+ssh://git@github.com/RUBclim/thermal-comfort
```

## Run the tests

On version 3.12 for example...

```bash
tox -e py312
```
